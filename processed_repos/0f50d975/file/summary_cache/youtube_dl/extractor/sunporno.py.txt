concise summary of the code

- Purpose
  - Implement a YouTube-DL extractor for sunporno.com videos (SunPornoIE). It fetches a video page, parses metadata, and collects all direct video formats for download.

- Key class
  - SunPornoIE(InfoExtractor)
    - _VALID_URL: RegExp matching both http(s) URLs for sunporno videos and embed pages:
      - http(s)://www.sunporno.com/videos/{id}
      - http(s)://embeds.sunporno.com/embed/{id}
    - _TESTS: Includes two tests:
      - A real page test with expected MD5 of the mp4 and an info_dict (id, ext, title, description, thumbnail, duration, age_limit).
      - An embed URL test with only_matching set to True.

- Main functionality (_real_extract)
  - Extracts the video_id from the URL using _match_id.
  - Downloads the video page HTML via _download_webpage.
  - Metadata extraction:
    - title: from the <title> tag.
    - description: from the meta description tag via _html_search_meta.
    - thumbnail: from a poster="..." attribute if present (fatal=False).
    - duration: robust extraction using two patterns:
      - itemprop="duration" pattern
      - a Duration: label pattern
      - Parsed with parse_duration if found.
    - view_count: extracted with a regex and converted to int_or_none (fatal=False).
    - comment_count: extracted with a regex and converted to int_or_none (fatal=False, default=None).
  - Video formats:
    - Scans the page for video sources via regex <(?:source|video) src="([^"]+)".
    - For each video URL:
      - Determine extension with determine_ext(video_url).
      - Build a format dict: url, format_id (extension), and quality (via a qualities function for ['mp4', 'flv']).
    - Sort formats with _sort_formats.
  - Returns a result dict containing:
    - id, title, description, thumbnail, duration, view_count, comment_count, formats
    - age_limit: 18

- Implementation details
  - Uses _HTML search helpers (_html_search_regex, _html_search_meta) with fatal flags to tolerate missing fields.
  - Uses int_or_none to safely convert numeric fields.
  - Qualities and determine_ext utilities help categorize and order formats (prefers mp4, flv).
